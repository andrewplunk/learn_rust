# Operators

## Arithmatic
+ - * / %

## Comparison
== != < > <= >=

## Logical
! && ||

## Bitwise
& | ^ << >>

## Assignment

```rust
let mut a = 2;

a += 5;
a -= 2;
a *= 2;
a /= 2;
a %= 5;
a &= 2;
a |= 2;
a ^= 2;
a <<= 1;
a >>= 2;
```

## Type Casting Operator

```rust
let a = 15;
let b = (a as f64) / 2.0;
```

## Borrowing and Dereference Operators 

& &mut *
